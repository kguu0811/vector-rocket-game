# Vector Rocket Game

## Description
A 2D vector-based rocket game inspired by ENG2005 concepts.

## Tools & Requirements
- macOS
- clang
- raylib
- Git

## Setup
xcode-select --install
brew install raylib

## Build
make

## Run
./build/main

## Controls
W/A/S/D: thrust
Shift: Boost

## Project structure
main.c
game.c
physics.c
object_list.c
